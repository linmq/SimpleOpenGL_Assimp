Visual Studio C++ 2015
Windows 10

The Demo's source code, the glut, the models come from Assimp-3.1
https://sourceforge.net/projects/assimp/files/assimp-3.1/

https://sourceforge.net/projects/assimp/files/assimp-3.1/
